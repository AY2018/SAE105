Four Jobs I would like to do when I graduate

1. Front end web Developer

Skills needed

  • Html
  • CSS
  • CSS Frameworks/Libraries (Bootstrap, Tailwind CSS, Sass)
  • JavaScript
  • JavaScript libraries and frameworks (React, Vue)
  • Databases (SQL, MySQL)

Front End Development relates to everything the user can see and interact with. For example, everything on this web site from the logo, the menu, to the images, videos, animations and layout of the page are managed by the front end developper, or in this case, me. To summarize, the front end developper is in charge of the look and feel of the website, on top of making sure that it looks good on all devices (phone, tablette, laptop).

I have loved building legos ever since I was young and I felt the same of satisfaction I get finishing a lego when I built my first website in HTML and CSS. Ever since then, it became a hobby of mine to just build websites and I hope to make it my job one day.

2. Back end web Developer

Skills needed

  • Python
  • (or) Java
  • (or) Node
  • (or) PHP

Back End Development refers to, the "behind the scenes of the website", the part of the application that the user do not see. Back end developers focus on the logic of the site, creating servers, and working with databases and API's (Application Programming Interfaces).

For example, the content and layout for the form in this page's footer is created on the client side, but when the users submits their information it gets processed on the server side.

I haven't started learning any of the back end languages but I'm looking forward to working on a more "technical" aspect of web developement. I have been good at maths since young and do generally well when it comes to logic so this might be an interesting job for me.

3. Full stack Engineer

Skills needed

  • All of the skills mentioned above

A Full Stack Developer is someone who works on both the front and back end of a web application. Working on both sides, he needs to have a good mastery of all the languages used whether in front end or back end. A Full Stack Developer also needs good problem solving skills as well as good debugging skills.

Full Stack Development looks like a good option further down my path as a Web Developer. I don't know for now if I will start as a Front end of Back end developer, but full stack development, being the combination of both, is an interesting career choice a few years after I star working.

4. Web Designer

Skills needed

  • Html
  • CSS
  • CSS Frameworks/Libraries (Bootstrap, Tailwind CSS, Sass)
  • JavaScript
  • Figma
  • Photoshop

If a Front End Developer is reponsible for building a website, then the Web Designer is responsible of for designing the layout, visual appearance and the usability of a website. A good web designer needs to have both the technicals skills (to propose the right design and not something that the prorgammer can't build) and the praphic skills.

A Web deisgner needs to be able to visualize how the web application will look like and how it will function (the code behind it)

I have built a good amount of websites and I enjoyed thinking about how to structure them, how to chooser the best colors to stay aligned with the brand's message, how to rearange the elements depending on the device. So Web Designer is possible alternative to my three prefered jobs that are more "programming oriented".
